## 1.使用方法

#### 1.克隆到本地
```bash
git clone https://github.com/aglorice/Raincourse.git
```

#### 2.进入目录
```bash
cd Raincourse
```
#### 3.安装依赖
```bash
pip install -r requirements.txt
```
#### 4.运行`main.py`
```bash
python main.py
```
